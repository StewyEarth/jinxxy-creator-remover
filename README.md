# JinxxyCreatorRemover
![Extension Preview](https://github.com/stewyearth/jinxxy-creator-remover/blob/main/github/images/Banner.png?raw=true)
## Features
- Hide posts from specific creators on Jinxxy.com
- Easy-to-use popup for managing your hidden creators list
- Works on both Firefox and Chrome
- All data is stored locally; no data is sent to any server

## Extension Store Links
- [Firefox Add-on](https://addons.mozilla.org/en-US/firefox/addon/jinxyy-creator-hider/)
- (Chrome Web Store link coming soon)

## Manual Installation
1. Download or clone this repository.
2. Go to your browser’s extensions/add-ons page.
3. Enable “Developer mode” (Chrome) or “Debug Add-ons” (Firefox).
4. Click “Load unpacked” (Chrome) or “Load Temporary Add-on” (Firefox) and select this folder.

## Usage
- Click the extension icon in your browser.
- Enter creator names (comma separated) to hide and click “Hide Creator/creators” or press Enter.
- The list of hidden creators will appear below.
- To remove a creator, click the “X” next to their name.
- Click “Reset” to clear your hidden creators list.

## Known issues
* Doesn't revert and show posts hidden without refreshing the page when a creator is removed from the list.
* Doesn't hide posts on the main page if the creator uses a store link that doesn't include the hidden name

## Privacy
- This extension does not collect, transmit, or share any personal information.
- All data is stored locally on your device.
